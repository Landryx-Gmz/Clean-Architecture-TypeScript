# Introducción

En este proyecto se implementarán dos tipos de arquitecturas basadas en Clean Architecture:

1. **Arquitectura Escalable**: Esta arquitectura está más orientada a ejemplos, proporcionando una visión general de cómo se puede aplicar Clean Architecture en un contexto más amplio y flexible.

2. **Arquitectura Simple**: Esta variante se centra en un proyecto paso a paso, través de la implementación de Clean Architecture de manera más directa y sencilla.

En cada tipo tendran algunos .md explicando todo su contenido, creación, estructuras, etc.
